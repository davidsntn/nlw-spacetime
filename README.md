## 💻 Projeto
Este é um projeto Web Responsivo de uma cápsula do tempo para exibir memórias em uma linha do tempo. 

## 🚀 Tecnologias
Este foi um projeto desenvolvido durante a NLW da Rocketseat com as seguintes tecnologias:

- HTML
- CSS
- Git e Github
